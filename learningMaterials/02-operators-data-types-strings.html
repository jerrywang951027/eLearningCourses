Notes

Page 1

Operators are the vocabulary of computation. Arithmetic operators include +, -, *, /, //, %, and **. Regular division returns a float, floor division discards the fractional part, modulo returns a remainder, and exponentiation raises a number to a power.

Comparison operators produce booleans: ==, !=, <, <=, >, and >=. Assignment = and equality == are different ideas. Boolean operators and, or, and not combine truth values and short-circuit when the result is already known.

Core types shape behavior. Integers are whole numbers, floats approximate decimals, strings store text, booleans store truth, and None marks absence. Conversion functions like int(), float(), str(), and bool() should be used deliberately at program boundaries.

Operator precedence controls evaluation order, but readable code often uses parentheses even when they are not required. Clarity beats cleverness while learning.

Page 2

Strings are immutable sequences of characters. They support indexing, slicing, membership checks, concatenation, repetition, and methods such as strip(), lower(), upper(), split(), and replace(). Methods return new strings rather than changing the original.

Slicing uses text[start:stop:step], with stop excluded. That exclusion rule also appears in range() and many Python APIs. It makes lengths and indexes consistent once you internalize it.

f-strings are the cleanest way to build readable text from values: f'Total: {total:.2f}'. Format specifiers control decimal places, percentages, alignment, commas, and other display details.

The most common beginner bug here is mixing text and numbers. '3' + '4' is '34', but 3 + 4 is 7. Convert input once, keep calculations numeric, then format final output as text.

Small Coding Challenges

1. Even Check

Write is_even(n) using modulo.

Reference implementation:

def is_even(n):
    return n % 2 == 0

2. Price Formatter

Return a dollar amount with two decimal places.

Reference implementation:

def format_price(amount):
    return f"${amount:.2f}"

3. Initials

Return initials from first and last names.

Reference implementation:

def initials(first, last):
    return f"{first[0].upper()}.{last[0].upper()}."

4. Clean Username

Trim spaces and lowercase a username.

Reference implementation:

def clean_username(username):
    return username.strip().lower()

5. Circle Area

Compute area from radius.

Reference implementation:

def circle_area(radius):
    return 3.14159 * radius ** 2
