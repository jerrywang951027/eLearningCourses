Notes

Page 1

Functions package behavior behind a name. A strong function has a clear job, meaningful parameters, and a predictable return value. print() displays information, while return sends data back to the caller.

Parameters are local names that receive arguments. Defaults are useful, but mutable defaults like [] should be avoided because the object is created once and reused across calls.

Scope describes where names are visible. Local names created inside a function are not available outside. Globals exist at module level, but changing them inside functions creates hidden dependencies.

Passing values in and returning values out is usually cleaner than reaching into outer state. It makes functions easier to test and reason about.

Page 2

Recursion is when a function calls itself to solve a smaller version of a problem. It needs a base case and a recursive step that moves toward that base case. Without a base case, recursion continues until Python raises an error.

Closures happen when an inner function remembers variables from an outer function. They are useful for factory functions that create configured behavior.

Lambda expressions create small anonymous functions. They are useful for short key functions, filters, and transformations, but multi-step logic should usually use def.

Higher order functions accept functions as arguments or return functions. sorted(key=...), map(), and filter() are common examples, though list comprehensions are often more readable than map/filter for beginners.

Small Coding Challenges

1. Safe Average

Return the average, or 0 for an empty list.

Reference implementation:

def average(numbers):
    if not numbers:
        return 0
    return sum(numbers) / len(numbers)

2. Factorial

Implement factorial recursively.

Reference implementation:

def factorial(n):
    if n < 0:
        raise ValueError("n must be non-negative")
    if n in (0, 1):
        return 1
    return n * factorial(n - 1)

3. Multiplier Closure

Return a function that multiplies by a fixed factor.

Reference implementation:

def make_multiplier(factor):
    def multiply(value):
        return value * factor
    return multiply

4. Sort People

Sort a list of dicts by age.

Reference implementation:

def sort_by_age(people):
    return sorted(people, key=lambda person: person["age"])

5. Discount

Apply a percent discount to a price.

Reference implementation:

def apply_discount(price, percent):
    return price * (1 - percent / 100)
