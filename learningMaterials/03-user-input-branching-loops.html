Page 1

Interactive programs read input, interpret it, and respond. input() always returns a string, so numeric input must be converted before arithmetic. Clear prompts tell the user exactly what kind of value is expected.

Branching uses if, elif, and else. Conditions are checked in order, and the first true branch in an if chain runs. Use elif for mutually exclusive choices and separate if statements for independent checks.

A for loop iterates over a sequence or range and is best when the number of repetitions or items is known. A while loop repeats while a condition remains true and is best when the stopping point depends on runtime behavior.

range(stop) starts at zero and excludes stop. range(start, stop, step) gives direct control over counting. Like slicing, the stop value is excluded.

Page 2

break exits the nearest loop immediately. continue skips the rest of the current iteration and moves to the next one. Both are useful, but too many jumps can make loop logic harder to read.

Infinite loops usually happen when the loop condition never moves toward False. In while loops, make sure some state changes each time or that a break condition is reachable.

Input validation combines loops and branches: ask for a value, check or convert it, and repeat until it is usable. Later, exception handling makes conversion validation safer.

Small challenge projects such as calculators, guessing games, and menu systems usually follow the same shape: initialize state, prompt, branch on choice, update state, and display feedback.

Small Coding Challenges

1. Classify Number

Return negative, zero, or positive.

Reference implementation:

def classify(n):
    if n < 0:
        return "negative"
    if n == 0:
        return "zero"
    return "positive"

2. Sum to N

Sum integers from 1 through n.

Reference implementation:

def sum_to_n(n):
    total = 0
    for value in range(1, n + 1):
        total += value
    return total

3. Count Vowels

Count vowels in a string.

Reference implementation:

def count_vowels(text):
    total = 0
    for ch in text.lower():
        if ch in "aeiou":
            total += 1
    return total

4. First Success

Return True if any attempt equals the secret.

Reference implementation:

def first_success(attempts, secret):
    for attempt in attempts:
        if attempt == secret:
            return True
    return False

5. Menu Choice

Map a, b, q to add, browse, quit.

Reference implementation:

def handle_choice(choice):
    if choice == "a":
        return "add"
    elif choice == "b":
        return "browse"
    elif choice == "q":
        return "quit"
    return "unknown"
